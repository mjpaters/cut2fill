## 1. TMR MRTS05 — Unbound Pavement Materials

**Document:** MRTS05 Unbound Pavements (July 2022)
**Source:** QLD Department of Transport and Main Roads
**URL:** https://www.tmr.qld.gov.au/business-industry/Technical-standards-publications/Specifications/5-Pavements-Subgrade-and-Surfacing

MRTS05 is the single most important material specification in QLD road construction. It covers all unbound granular pavement materials — natural, quarried, and recycled. Materials are classified into 4 Types with subtypes.

### 1.1 Type 1 — High Standard Granular (HSG)

| Code | Name | Plain English | Typical Use |
|------|------|---------------|-------------|
| **Type 1 (HSG)** | High Standard Granular | Premium crushed rock with strict hardness, durability, and grading requirements. Tightest tolerances of all types. | High-traffic road bases (state highways, motorways). Upper pavement layers on TMR-controlled roads. |

**Key properties:**
- Hard, sound, durable aggregate — must not break down in service
- Strict Los Angeles Abrasion and Wet/Dry Strength Variation limits
- Quarry must hold TMR registration (QRS certification required)
- Based on historically specified Subtype 1.1 materials
- Most expensive unbound pavement material

### 1.2 Type 2 — Standard Crushed Rock Pavements

The workhorse of QLD road construction. Five subtypes with decreasing quality requirements.

| Code | Name | Plain English | Liquid Limit | Plasticity Index | Typical Use |
|------|------|---------------|:---:|:---:|-------------|
| **Type 2.1** | Subtype 2.1 | Top-quality crushed rock. Very low plasticity, tight grading. | max 25 | max 6 | Base course on high-traffic roads. Upper pavement layers. |
| **Type 2.2** | Subtype 2.2 | High-quality crushed rock. Same plasticity limits as 2.1 but slightly relaxed grading options. | max 25 | max 6 | Base course, upper subbase. Similar to 2.1 with more grading flexibility. |
| **Type 2.3** | Subtype 2.3 | Good-quality crushed rock. Slightly higher plasticity allowed. | max 28 | max 8 | Base or upper subbase on moderate-traffic roads. Most commonly specified Type 2 material. |
| **Type 2.4** | Subtype 2.4 | Moderate-quality crushed rock. More plasticity tolerance. | max 35 | max 12 | Lower subbase layers. Lower-traffic roads. |
| **Type 2.5** | Subtype 2.5 | Lower-quality crushed rock or select natural gravel. Most relaxed Type 2 requirements. | max 40 | max 14 | Subbase, selected fill above subgrade. Lower-order roads and council works. |

**Grading envelopes:**
- Subtypes 2.1, 2.2, 2.3: Accept Grading B, C, or Modified C
- Subtypes 2.4, 2.5: Additional Grading D option for base or upper subbase layers
- All subtypes require quarry assessment and TMR QRS registration (except 2.5 which can use some natural sources)

**What quarries supply:** When a quarry is "TMR registered for Type 2.3", it means that specific quarry's product has been tested and certified by TMR to meet Type 2.3 requirements. The registration is product-specific and quarry-specific.

### 1.3 Type 3 — Natural Gravel / Lower-Quality Pavements

| Code | Name | Plain English | Typical Use |
|------|------|---------------|-------------|
| **Type 3.1** | Subtype 3.1 | Good natural gravel. Strength principally assessed by soaked CBR. | Subbase on sealed roads. |
| **Type 3.2** | Subtype 3.2 | Moderate natural gravel. | Subbase, lower pavement layers. |
| **Type 3.3** | Subtype 3.3 | Standard natural gravel. Common regional material. | Subbase on lower-order roads. Unsealed road wearing course. |
| **Type 3.4** | Subtype 3.4 | Lower-quality natural gravel. | Fill above subgrade improvement. Lower-order unsealed roads. |
| **Type 3.5** | Subtype 3.5 | Lowest-quality pavement-grade gravel. Most relaxed requirements. | Selected subgrade, capping layer. Unsealed roads in remote areas. |

**Key differences from Type 2:**
- CBR (California Bearing Ratio) is the primary strength measure (Table 7.3.5)
- Quarry registration not required for all subtypes (3.5 exempt)
- Natural gravel sources accepted (ridge gravels, laterite, ironstone, duricrust)
- More suited to regional/remote QLD where crushed rock is expensive to transport

### 1.4 Type 4 — Recycled and Non-Standard Materials

| Code | Name | Plain English | Typical Use |
|------|------|---------------|-------------|
| **Type 4** | Recycled/Non-Standard Pavement Material | Recycled concrete, recycled asphalt (RAP), industrial by-products (slag, fly ash), or other non-standard materials assessed on a project-specific basis. | Subbase, selected fill, lower pavement layers. Increasing use in sustainability-focused projects. |

**Key requirements:**
- Must meet specific grading, plasticity, and strength criteria per project
- Requires a Recycled Material Assessment (QRS5)
- Environmental compliance (leachate testing, contaminant limits)
- TMR is progressively encouraging Type 4 use

## 2. TMR MRTS04 — General Earthworks (Fill Classifications)

**Document:** MRTS04 General Earthworks (March 2025)
**Source:** QLD Department of Transport and Main Roads
**URL:** https://www.tmr.qld.gov.au/business-industry/Technical-standards-publications/Specifications/3-Roadworks-Drainage-Culverts-and-Geotechnical

MRTS04 governs all earthworks in QLD road and bridge construction. Fill material is classified into distinct classes based on quality, with better materials used higher in the embankment.

### 2.1 Fill Classes

| Code | Name | Plain English | Key Properties | Typical Use |
|------|------|---------------|----------------|-------------|
| **Class A1** | Class A1 Fill | Premium earth fill. Low plasticity, good compaction characteristics, non-dispersive. Becoming scarce across QLD. | Strict PSD, PI, LS, CBR, swell limits | Top 500mm of embankments. Behind structures. Bridge approaches. |
| **Class B** | Class B Fill | Standard earth fill. Good quality but more relaxed than A1. TMR now specifies Class B in preference to A1 where both meet performance requirements (to conserve A1 supply). | Moderate PSD, PI, LS, CBR limits | General embankment construction (upper zones). Standard fill behind structures. |
| **Class C** | Class C Fill | General earth fill. Acceptable quality for bulk fill. Higher plasticity and lower strength tolerance. | Relaxed PI, LS, CBR limits | Lower embankment zones. Non-structural fill areas. |
| **Class D** | Class D Fill | Lower-quality fill. Marginal material acceptable for specific locations. | Minimal requirements | Deep embankment zones (lower third). Areas away from pavement. |
| **Rock Fill** | Rock Fill | Material predominantly comprising rock fragments, used where high-strength fill is required or available from cut. | Max particle size, compaction by rolling | Rock embankments. Structural fill where rock is the cut material. |

**Key assessment properties:**
- Particle Size Distribution (PSD)
- California Bearing Ratio (CBR) — soaked
- Plasticity Index (PI)
- Weighted Plasticity Index (WPI)
- Linear Shrinkage (LS)
- Swell
- Dispersivity (Emerson Class, Pinhole test)

**Important policy note:** TMR Technical Note 216 (November 2025) states that Class A1 material is becoming increasingly constrained. Where both Class A1 and Class B meet performance requirements, Class B must now be specified in preference. This has supply chain implications for the Cut2Fill platform.

### 2.2 Other MRTS04 Material Categories

| Category | Plain English | Standard |
|----------|---------------|----------|
| **Select Fill** | Material meeting specific engineering requirements for placement directly beneath the pavement (subgrade improvement). | MRTS04 — properties specified per project. |
| **Imported Fill** | Fill material brought to site from an external source (as opposed to material won from on-site cut). | MRTS04 — must meet the class requirements for its placement zone. |
| **Unsuitable Material** | Material not meeting any fill class requirements — must be wasted or treated. Includes highly plastic clays, organic soils, contaminated material. | MRTS04 — identified during construction and disposed of. |

## 4. TMR Quarry Registration System (QRS)

**Source:** TMR Geotechnical Unit
**URL:** https://www.tmr.qld.gov.au/business-industry/business-with-us/approved-products-and-suppliers/pavements-materials-and-geotechnical

TMR's QRS oversees approximately 440 registered quarries that supply materials to TMR, local governments, and private industry.

### 4.1 QRS Documents

| Document | Purpose |
|----------|---------|
| **QRS1** | Quarry Registration System outline — overarching requirements |
| **QRS2** | Quarry Registration System — detailed registration process |
| **QRS5** | Preparing a Recycled Material Assessment — for Type 4 materials |

### 4.2 What Registration Means

- A quarry is registered for **specific material types** (e.g., "Type 2.3 Grading C")
- Registration is product-specific and quarry-specific
- The quarry operator carries out source rock testing per a Registered Overall Testing Frequency Schedule
- Results are forwarded to the TMR Technical Manager (QRS) for ongoing compliance
- Registered quarries can supply to TMR projects, council projects, and private works

| Type 2.1 | High-quality crushed rock base |

## 5. Environmental / Contamination Classifications

### 5.1 NEPM Soil Classifications (National, Applied in QLD)

**Document:** National Environment Protection (Assessment of Site Contamination) Measure
**Source:** National Environment Protection Council (NEPC)
**Applied via:** QLD Environmental Protection Act 1994

| Classification | Code | Plain English | Testing Required | Disposal/Reuse |
|---------------|------|---------------|-----------------|----------------|
| **Virgin Excavated Natural Material** | VENM | Soil that has never been subject to any contaminating activity. Natural background concentrations only. The "cleanest" classification. | Visual assessment + site history review. No lab testing if provenance is clear. | Unrestricted reuse anywhere. |
| **Excavated Natural Material** | ENM | Soil from non-contaminated sites that may have slightly elevated but non-hazardous concentrations. | Laboratory testing against NEPM criteria required. | Reuse with conditions (e.g., not in sensitive land uses). |
| **Contaminated Tier 1** | CT-1 | Soil exceeding NEPM Health Investigation Levels (HILs) for commercial/industrial use but below hazardous thresholds. | Full laboratory suite per NEPM (metals, TPH, PAH, OCP, VOC etc.). | Licensed facility only. Must use licensed waste transport. |
| **Contaminated Tier 2** | CT-2 | Soil exceeding NEPM HILs significantly. May contain hazardous substances. | Extended laboratory suite. Possibly includes TCLP leachate testing. | Licensed high-security facility only. Regulated waste transport (ERA 57). |

### 5.2 QLD Environmental Protection Act Fill Categories

**Document:** Environmental Protection Act 1994 (Qld) + Environmental Protection Regulation 2019
**Source:** QLD Department of Environment and Science

| Category | Plain English | Regulatory Trigger |
|----------|---------------|--------------------|
| **Clean Earth / Clean Fill** | Uncontaminated soil suitable for direct reuse. Note: QLD has NO End of Waste (EOW) code for clean fill — it is legally "waste" once excavated, even if perfectly clean. | General Environmental Duty (s.319). Waste levy applies if sent to landfill ($125/t in 2025-26, metro zone). Day cover exempt. |
| **Acid Sulfate Soil** | Soil containing iron sulfides that produce sulfuric acid when exposed to air. Common in coastal QLD below 5m AHD. | ASSMP required. SPP trigger: excavation >=100m3 at/below 5m AHD. |
| **Contaminated Soil** | Soil from EMR/CLR-listed land or soil exceeding NEPM investigation levels. | Soil Disposal Permit required (EP Act s.424). |
| **Regulated Waste** | Waste listed in Schedule 7 of the EP Regulation. Includes some contaminated soils, asbestos-containing material, PFAS-impacted soil. | Environmental Authority (ERA 57) for transport. Waste tracking certificates. |
| **Prescribed Contaminant Soil** | Soil containing prescribed contaminants (e.g., PFAS, heavy metals above thresholds). | Unlawful deposit carries 4,500 penalty units or 5 years imprisonment (s.440ZG). |

### 5.3 Asbestos-Related Classifications

| Classification | Plain English | Standard |
|---------------|---------------|---------|
| **Asbestos-Free** | Material confirmed free of asbestos through inspection or testing. | Workplace Health and Safety Act 2011 + Reg. |
| **ACM (Asbestos Containing Material)** | Material containing bonded (non-friable) asbestos. Common in pre-1990 demolition material. | Must be handled by licensed removalist. Disposed at licensed facility. |
| **Friable Asbestos** | Asbestos that can be crumbled by hand. Highest risk category. | Class A licensed removalist only. Specialist disposal. |

## 6. Earthworks Material Types (General Civil Construction)

These are the practical material categories used across QLD civil/road construction, cross-referencing the TMR specifications.

| Material Type | TMR Spec Reference | Plain English | Typical Source | Typical Use |
|--------------|-------------------|---------------|----------------|-------------|
| **General Fill** | MRTS04 Class B/C/D | Bulk earth fill — clay, silt, sand, gravel mix. The most commonly moved material in construction. | Site cut, borrow pits, imported fill suppliers | Embankment construction, site levelling, backfill |
| **Structural Fill** | MRTS04 Class A1 | High-quality fill placed in zones requiring engineering performance (behind retaining walls, bridge abutments). | Selected cut, quarry-supplied material | Behind structures, bridge approaches, upper embankment |
| **Select Fill / Subgrade Improvement** | MRTS04 + MRTS05 | Specific-quality material placed directly beneath the pavement to improve subgrade performance. | Quarries, screened site material | Top of embankment (subgrade zone), under pavements |
| **Rock Fill** | MRTS04 Rock Fill | Fill comprising predominantly rock fragments from blasting or ripping. | Site rock cut, quarry overburden | Rock embankments, drainage blankets, stable fill in wet areas |
| **Sand Fill** | MRTS04 (as applicable) | Clean sand used for fill, drainage, bedding. | Sand quarries, river extraction (licensed) | Pipe bedding, drainage layers, sand blankets, beach nourishment |
| **Topsoil** | MRTS04 (stripping/respread) | Organic-rich surface soil for revegetation. Not a structural material. | Site stripping, topsoil suppliers | Landscaping, batter treatments, revegetation, erosion control |
| **Drainage Material** | MRTS05 / MRTS03 | Sized aggregate (clean, free-draining) for subsurface drainage systems. | Quarries | Subsoil drains, filter layers, drainage blankets |
| **Capping Layer** | MRTS05 Type 3.5 or better | Material placed on top of prepared subgrade to protect it and provide a working platform. | Quarries, gravel pits | Layer between subgrade and pavement, working platform |
| **Stabilised Material** | MRTS07B/07C/08/09/10 | Existing or imported material mixed with a binder (cement, lime, bitumen) to improve strength. | On-site material + binder, or plant-mixed | Pavement rehabilitation, subgrade improvement |

| Material | Description | QLD Requirements |
|----------|-------------|-----------------|

## 8. Concrete Material Types

### 8.1 Normal Class (N Grade) — AS 1379

**Document:** AS 1379-2007 (R2017) Specification and Supply of Concrete
**Source:** Standards Australia

Normal-class concrete is specified by strength grade alone. The supplier determines the mix design.

| Grade | Strength (MPa) | Plain English | Typical Use |
|-------|:-:|---------------|-------------|
| **N20** | 20 | Low-strength concrete. Residential/light-duty. | Residential footings, pathways, house slabs (lightly loaded), bedding. |
| **N25** | 25 | Standard residential concrete. | House slabs, driveways, footpaths, light commercial footings. |
| **N32** | 32 | Standard structural concrete. Most commonly specified in QLD commercial/civil works. | Structural slabs, beams, columns, retaining walls, bridge components. |
| **N40** | 40 | High-strength structural concrete. | Heavily loaded structures, multi-storey buildings, bridge decks, precast elements. |
| **N50** | 50 | Very high-strength concrete. Specialist applications. | High-rise buildings, prestressed elements, marine structures. |

### 8.2 Special Class (S Grade) — AS 1379

Special-class concrete has additional specified properties beyond just strength (e.g., durability, shrinkage, permeability). The purchaser specifies the mix requirements.

| Grade | Strength (MPa) | W/C Ratio | Min Cement (kg/m3) | Plain English | Typical Use |
|-------|:-:|:-:|:-:|---------------|-------------|
| **S20** | 20 | 0.56 | 300 | Durable low-strength concrete. | Kerb and channel, minor drainage structures. |
| **S25** | 25 | 0.52 | 320 | Durable standard concrete. | Footpaths, drains, non-structural elements in exposed environments. |
| **S32** | 32 | 0.50 | 340 | Durable structural concrete. | External structural elements, car parks, water-retaining structures. |
| **S40** | 40 | 0.45 | 380 | High-strength durable concrete. | Bridge decks, marine piles, water/wastewater structures, TMR structural works. |
| **S50** | 50 | 0.40 | 420 | Very high-strength durable concrete. | Prestressed bridge girders, high-performance structures. |
| **S65** | 65 | 0.38 | 450 | Ultra-high-strength durable concrete. Specialist specification. | Super-T girders, post-tensioned elements, high-rise cores. |

### 8.3 Other Concrete Products

| Product | Plain English | Standard | Typical Use |
|---------|---------------|----------|-------------|
| **Lean Mix / Sub-base Concrete** | Low-strength concrete (typically 5-10 MPa) used as a pavement sub-layer. | MRTS39 | Subbase under concrete pavements, bedding under pipes. |
| **Flowable Fill / CLSM** | Very low-strength (1-3 MPa) self-levelling concrete used as fill. Controlled Low Strength Material. | Project-specific | Backfill around utilities, void filling, abandoned pipe filling. |

## 9. Water Types for Construction

| Type | Plain English | Source | Typical Use | QLD Requirements |
|------|---------------|--------|-------------|-----------------|
| **Potable Water** | Town water from council/SEQ Water supply. Meets Australian Drinking Water Guidelines. | Council mains, standpipes | Concrete mixing, dust suppression, compaction moisture, curing | Metered standpipe permit from council. Most expensive option. |
| **Raw / River Water** | Untreated water from rivers, creeks, dams. Variable quality. | Licensed extraction points, farm dams | Dust suppression, compaction, non-structural concrete (if tested) | Water extraction licence from DRDMW required for watercourse extraction. Allocation limits apply. |
| **Recycled Water** | Treated wastewater/effluent meeting Class A, B, or C quality. | Council recycled water schemes (e.g., SEQ Western Corridor) | Dust suppression, compaction, irrigation, some concrete mixing (Class A+) | Must comply with QLD Water Supply (Safety and Reliability) Act 2008. Class determines permitted uses. |
| **Bore Water / Groundwater** | Water extracted from underground aquifers. | Registered bores | Dust suppression, compaction, concrete mixing (if tested and suitable) | Bore registration + water licence from DRDMW. Salinity and chemical testing for concrete use (AS 1379 limits). |
| **Stormwater / Captured Runoff** | Rainwater captured on-site from detention basins or runoff. | Site detention basins, rainwater tanks | Dust suppression, compaction | Must manage sediment. May need to test for suitability if used in concrete or stabilisation. |
| **Saline Water** | Water with elevated salt content (from coastal bores, tidal areas). | Coastal bores, estuarine extraction | Very limited — dust suppression only in some cases | NOT suitable for concrete (chloride attack on reinforcement). NOT suitable for cement-stabilised materials. AS 1379 chloride limits apply. |

**Water quality for concrete (AS 1379):**
- pH: 5.0-8.5
- Chlorides: <500mg/L (prestressed), <1000mg/L (reinforced)
- Sulfates: <1000mg/L
- Total dissolved solids: <2000mg/L

## 10. QLD Council Specification Practices

### 10.1 Do Councils Use TMR Specs?

**Yes — almost universally.** QLD councils adopt TMR MRTS specifications as their base standard, with local amendments via annexures.

| Framework | Councils | Approach |
|-----------|----------|----------|
| **TMR MRTS (direct adoption)** | Most SEQ councils (BCC, GCCC, LCC, ICC, MBRC, SCRC, RCC) | Adopt MRTS specs with project-specific annexures. Development approval conditions reference MRTS. |
| **FNQROC Development Manual** | 10 Far North QLD councils (Cairns, Douglas, Mareeba, Tablelands, Cassowary Coast, Hinchinbrook, etc.) | Regional manual that references MRTS05, MRTS04 directly. Adds regional amendments for tropical conditions. Version 9 (2023) current. |
| **IPWEAQ Standard Drawings** | Most QLD councils | Standard engineering drawings for local government. 96 drawings covering roads, drainage, parks. Referenced in council planning schemes alongside MRTS. |
| **IPWEAQ LORDG** | Most QLD councils | Lower Order Road Design Guidelines. Covers >70% of the local road network. References TMR specs for material requirements but allows risk-based relaxations for low-traffic roads. |

### 10.2 Key Council Differences

| Council | Notable Differences |
|---------|-------------------|
| **Brisbane City Council** | Has its own Subdivision and Development Guidelines. Generally stricter than TMR for development works. Additional stormwater quality requirements. |
| **Gold Coast City Council** | "Our Living City" planning scheme. Land development guidelines reference MRTS. Additional coastal/canal estate fill requirements. |
| **Sunshine Coast Regional Council** | Engineering standards reference TMR. Additional acid sulfate soil considerations in coastal areas. |
| **Regional councils** | Often specify Type 2.5 or Type 3.3 as base course for lower-order roads (cost-driven — crushed rock cartage distances are long in regional QLD). |

### 10.3 What This Means for Cut2Fill

A material listed as "Type 2.3" on Cut2Fill is universally understood across QLD — whether the buyer is TMR, a council, or a private developer. The TMR classification system is the common language.

## 12. Platform Integration Recommendations

### 12.1 Smart Material Selector Behaviour

When a user selects a material category, the platform should:

1. **Show the TMR classification** as the primary identifier (Type 2.3, Class B, N32, etc.)
2. **Show plain English description** so non-technical users understand
3. **Show compliance tooltip** with regulatory requirements for that material
4. **Auto-suggest matching materials** — e.g., if someone has Type 2.3 available, match with anyone wanting Type 2.3 or lower
5. **Show substitution guidance** — e.g., "Type 2.3 can also be used where Type 2.4 or 2.5 is specified"

### 12.2 Material Compatibility Matrix

Higher-spec material can generally substitute for lower-spec material:

```
Type 1 (HSG) > Type 2.1 > Type 2.2 > Type 2.3 > Type 2.4 > Type 2.5
Class A1 > Class B > Class C > Class D
N50 > N40 > N32 > N25 > N20
```

This means the matching engine can offer "upgrades" — if someone needs Type 2.5 and there's Type 2.3 available nearby, that's a valid match.

### 12.3 Quarry Product Mapping

Each quarry in the Cut2Fill database should list its TMR-registered product types. When a user searches for "Type 2.3", the platform shows:
- Active listings of Type 2.3 material
- Quarries registered to supply Type 2.3
- Listings of Type 2.1 or 2.2 (which also satisfy a 2.3 requirement)
